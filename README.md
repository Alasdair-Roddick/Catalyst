# create-catalyst

Scaffold a new [Catalyst](https://github.com/Alasdair-Roddick/catalyst) project.

## Usage

```bash
bunx create-catalyst my-app
cd my-app
bun run dev
```

## What You Get

- Bun-native SSR with React
- File-based routing
- Nested layouts
- Tailwind CSS
- Live reload
- Zero config

## Local Development

To use locally without publishing:

```bash
bun /path/to/create-catalyst/index.ts my-app
```
